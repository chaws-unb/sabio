sabio
=====

Sabio is capable to compile and correct the Common Weakness Enumeration (CWE) of source code written in C.

Folder Structure
=====

bin/ 	- Final binary
inc/ 	- headers
lex/ 	- Flex files
obj/ 	- Object files
others/
build/ 	- Use this folder for build final application.
par/ 	- Bison files
src/ 	- C sources
